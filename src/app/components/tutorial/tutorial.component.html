Searching promiscuous proteins in ProtMiscuity

There are many ways to find your protein in ProtMiscuity

Search by Protein Name

The search is by default set by protein or Uniprot id. The user can write UniProt id or the protein name in the the field and click the search button.

Search a protein
Advanced search
  • By organism

    • By clicking this option, the user will obtain all the promiscuous proteins related with the organism of interest.

  • By reaction

    • By clicking this option, the user will obtain all the proteins having this promiscuous or canonic reaction.

After a search, the results page show a card where you can press in order to get the all the information available of this protein.

Visualazing protein data

All the information is collapsed and listed in five items in which the user can search for the specific information of interest: general information, sequence and active sites information, sctructural information and visualization and conformational dyversity and external links.

Structures visualization

Structural visualization of all PDB related with the protein of interest. All the PDBs are represented in clckable cards. The user can acces the visualizator by clicking on the cards and customize the view by selecting the style.
